# Print the `ptblank_agent` object

This function will allow the agent object to print a useful HTML-based
report.

## Usage

``` r
# S3 method for class 'ptblank_agent'
print(x, view = interactive(), ...)
```

## Arguments

- x:

  An object of class `ptblank_agent`.

- view:

  The value for [`print()`](https://rdrr.io/r/base/print.html)s `browse`
  argument.

- ...:

  Any additional parameters.
