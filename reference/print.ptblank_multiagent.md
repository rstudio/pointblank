# Print the `ptblank_multiagent` object

This function will allow the multiagent object to print a useful
HTML-based report.

## Usage

``` r
# S3 method for class 'ptblank_multiagent'
print(x, view = interactive(), ...)
```

## Arguments

- x:

  An object of class `ptblank_multiagent`.

- view:

  The value for [`print()`](https://rdrr.io/r/base/print.html)s `browse`
  argument.

- ...:

  Any additional parameters.
