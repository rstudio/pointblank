# Print the `ptblank_informant` object

This function will allow the informant object to print a useful
HTML-based report.

## Usage

``` r
# S3 method for class 'ptblank_informant'
print(x, view = interactive(), ...)
```

## Arguments

- x:

  An informant object of class `ptblank_informant`.

- view:

  The value for [`print()`](https://rdrr.io/r/base/print.html)s `browse`
  argument.

- ...:

  Any additional parameters.
