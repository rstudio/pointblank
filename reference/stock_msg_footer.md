# Provide simple email message body components: footer

The `stock_msg_footer()` function simply provides some stock text for an
email message sent via
[`email_blast()`](https://rstudio.github.io/pointblank/reference/email_blast.md)
or obtained as a standalone object through
[`email_create()`](https://rstudio.github.io/pointblank/reference/email_create.md).

## Usage

``` r
stock_msg_footer()
```

## Value

Text suitable for the `msg_footer` argument of
[`email_blast()`](https://rstudio.github.io/pointblank/reference/email_blast.md)
and
[`email_create()`](https://rstudio.github.io/pointblank/reference/email_create.md).
